═════════════════════════════════════════════════════════════════

   ├─ _id: ObjectId (unique identifier)
   ├─ nom: String (last name)
   ├─ prenom: String (first name)
   ├─ email: String | null
   ├─ telephone: String | null
   ├─ code_pays: String | null (country code)

   ├─ _id: ObjectId
   ├─ type_piece: String (document type)
   ├─ nom: String (owner name)

   ├─ date_naissance: String (birth date)
   ├─ lieu_naissance: String (birth place)
   ├─ telephone_declarant: String (finder phone)
   ├─ photo_path: String (image path)
   ├─ user_id: ObjectId (reference to user)
   └─ date_declaration: DateTime

🎯 USER JOURNEY
═════════════════════════════════════════════════════════════════

   FINDER WORKFLOW
   ───────────────
   1. Click "Créer un compte" → /register
   2. Fill registration form
   3. Create account → /registration-success
   4. Click "Déclarer un document" → /declare
   5. Select user account
   6. Fill document details
   7. Upload document image
   8. Submit → /declaration-success

   SEARCHER WORKFLOW
   ──────────────────
   1. Click "Rechercher un document" → /search
   2. Fill search criteria
   3. Click Search
   4. View results with contact info
   5. Contact finder directly (phone)
