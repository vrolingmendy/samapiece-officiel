ROUTES DISPONIBLES
==================

GET  /                      → Page accueil
GET  /register              → Formulaire enregistrement
POST /register              → Soumettre enregistrement
GET  /registration-success  → Confirmation enregistrement
GET  /declare               → Formulaire déclaration
POST /declare               → Soumettre déclaration
GET  /declaration-success   → Confirmation déclaration
GET  /search                → Formulaire recherche
POST /search                → Soumettre recherche
GET  /uploads/<filename>    → Images uploadées

MODÈLES DE DONNÉES
==================

Collection users:
- _id (ObjectId)
- nom, prenom (String)
- email, telephone (String | null)
- code_pays (String | null)
- date_creation (DateTime)

Collection documents:
- _id (ObjectId)
- type_piece, nom, prenom (String)
- date_naissance, lieu_naissance (String)
- telephone_declarant (String)
- photo_path (String - chemin relatif)
- user_id (ObjectId - référence)
- date_declaration (DateTime)
